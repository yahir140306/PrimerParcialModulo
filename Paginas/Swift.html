Lenguaje de programación Swift

Swift es un lenguaje de programación de código abierto para el desarrollo de aplicaciones de iOS, macOS, watchOS y tvOS. Fue desarrollado por Apple en el año 2014 como un reemplazo más seguro y rápido para Objective-C.

Programas en Swift

A continuación se muestran algunos programas básicos en Swift:

Hola mundo

Este programa simplemente imprime "Hola mundo" en la consola:


  print("Hola mundo")

Suma de dos números

Este programa toma dos números ingresados por el usuario y muestra la suma:


  print("Ingrese el primer número:")
  let num1 = Int(readLine()!)!
  
  print("Ingrese el segundo número:")
  let num2 = Int(readLine()!)!
  
  let suma = num1 + num2
  
  print("La suma de los dos números es: \(suma)")
