Lenguaje de Programación SQL

SQL (Structured Query Language) es un lenguaje de programación utilizado para manejar y manipular bases de datos relacionales. Fue creado en la década de 1970 por IBM, y es utilizado por una amplia variedad de sistemas de gestión de bases de datos, como Oracle, MySQL, Microsoft SQL Server y PostgreSQL.

Ejemplo de Programa en SQL

A continuación se muestra un ejemplo de código SQL que crea una tabla de empleados:

      
CREATE TABLE empleados (
    id INT PRIMARY KEY,
    nombre VARCHAR(50),
    apellido VARCHAR(50),
    fecha_nacimiento DATE,
    salario FLOAT
);

Este código crea una tabla llamada "empleados" con cinco columnas: "id", "nombre", "apellido", "fecha_nacimiento" y "salario". La columna "id" es la clave principal de la tabla.

Características de SQL

Las características principales de SQL incluyen:

  • Lenguaje de programación para bases de datos relacionales
  • Permite la manipulación de datos en las bases de datos
  • Permite la creación, modificación y eliminación de tablas y otros objetos en la base de datos
  • Permite la gestión de permisos y usuarios en la base de datos
  • Soporta la ejecución de consultas y filtros para recuperar datos específicos de la base de datos
  • Permite la creación de vistas, procedimientos almacenados y funciones en la base de datos

Tabla de Comandos Básicos de SQL

Comando Descripción
SELECT Recupera datos de la base de datos
INSERT Inserta nuevos datos en la base de datos
UPDATE Actualiza datos existentes en la base de datos
DELETE Elimina datos existentes en la base de datos
CREATE Crea una tabla o un objeto en la base de datos
ALTER Modifica una tabla o un objeto existente en la base de datos
DROP Elimina una tabla o un objeto existente en la base de datos
