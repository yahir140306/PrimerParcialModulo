Lenguaje de Programación C++

El lenguaje de programación C++ es un lenguaje de programación de propósito general que se utiliza para crear programas de software. C++ es un lenguaje orientado a objetos que se basa en el lenguaje C y que agrega características adicionales para la programación orientada a objetos.

Ejemplo de Programa en C++

A continuación se muestra un programa simple en C++ que imprime "Hola Mundo" en la pantalla:

      
#include 
int main() {
std::cout << "Hola Mundo!" << std::endl;
return 0;
}

Características de C++

Las características principales de C++ incluyen:

  • Lenguaje de programación de propósito general
  • Orientado a objetos
  • Compilado
  • Permite programación genérica
  • Permite la programación de bajo nivel
