Lenguaje de Programación C#

C# (pronunciado "C Sharp") es un lenguaje de programación moderno, orientado a objetos y de propósito general desarrollado por Microsoft. Fue creado en el año 2000 y es una de las principales opciones para desarrollar aplicaciones para la plataforma .NET.

Ejemplo de Programa en C#

A continuación se muestra un ejemplo de código C# que muestra un mensaje de saludo en la consola:

      
using System;
class Program
{
static void Main(string[] args)
{
Console.WriteLine("¡Hola, Mundo!");
}
}

Este código muestra el mensaje "¡Hola, Mundo!" en la consola cuando se ejecuta.

Características de C#

Las características principales de C# incluyen:

  • Lenguaje de programación moderno, orientado a objetos y de propósito general
  • Creado por Microsoft
  • Es una de las principales opciones para desarrollar aplicaciones para la plataforma .NET
  • Es un lenguaje tipado estáticamente, lo que significa que los tipos de datos deben ser especificados en tiempo de compilación
  • Permite la programación orientada a objetos, el manejo de excepciones, la programación asincrónica, entre otros
