Lenguaje de Programación HTML

HTML es un lenguaje de marcado utilizado para crear páginas web. El nombre HTML significa "HyperText Markup Language" y se utiliza para estructurar el contenido de una página web en forma de texto y otros elementos multimedia como imágenes, videos y audio.

Ejemplo de Programa en HTML

A continuación se muestra un programa simple en HTML que crea una lista no ordenada:

      
<!DOCTYPE html>
<html>
  <head>
    <title>Mi Lista</title>
  </head>
  <body>
    <h1>Mi Lista</h1>
    <ul>
      <li>Elemento 1</li>
      <li>Elemento 2</li>
      <li>Elemento 3</li>
    </ul>
  </body>
</html>

Este código creará una lista no ordenada con tres elementos.

Características de HTML

Las características principales de HTML incluyen:

  • Lenguaje de marcado utilizado para crear páginas web
  • Se utiliza para estructurar el contenido de una página web
  • Permite la incorporación de otros elementos multimedia
  • Es fácil de aprender y utilizar
